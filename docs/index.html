The Problem

Every AI-generated plan is a lie.

Not a malicious lie. Not a deliberate deception. But a lie nonetheless — a polished module list that mimics the shape of understanding without any of the substance. "User management. Payment system. Recommendation engine." It looks structured. It looks comprehensive. It is a shell.

The plan didn't warn you because the plan never saw the wall. It wasn't looking.

Decomposer exists to force the wall into visibility before you walk into it.

The fundamental unit of progress is the conversion of
one unknown unknown into a known unknown.

The Protocol

Five Steps. No AI Required.

The protocol runs in your head, on a whiteboard, or in a conversation. The AI skill is a convenience, not a requirement.

01

Honesty Fuse

Before any analysis, state three things: what you can actually engage with, where you are blind, and how this thing most likely dies.

HARD GATE
02

Uncertainty Mapping

Classify every element into three zones. Green = Known-Known, Yellow = Known-Unknown, Red = Unknown-Unknown. If Green exceeds 70%, the map is dishonest.

03

Hierarchical Decomposition

Transform the flat map into a cognitive dependency tree. This is not functional decomposition — ask "What must I know before I can know X?"

04

Error Budget Assignment

For each node: Confidence (how fast would failure surface?), Survival Condition (Alive if / Dead if), Lethality (what dies with it).

05

Anti-Shell Self-Check

Five hard checks: Survival Condition, Jargon, Uncertainty Ratio, Honesty, Actionability. If any fails, return to the corresponding step. Repeat until all pass.

The Only Metrics

Honesty

If the output is dishonest, it is worthless. The protocol begins with a confession of limits, not a plan.

Actionability

If it is honest but you cannot act on it, it is incomplete. Every node has a concrete first step.

"If the best we can deliver is three honest nodes with survival conditions — three pieces of the problem that are genuinely understood and genuinely actionable — that is superior to a fifty-page plan where every module is a placeholder in disguise."
