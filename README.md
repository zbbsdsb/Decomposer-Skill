# Decomposer

A cognitive operation protocol for transforming unknown unknowns into known unknowns.

Honest. Actionable. Domain-agnostic.

## What This Is

Not a project management framework. Not an AI tool. Not a template library.

A protocol — a repeatable, teachable five-step process for decomposing any large, ambiguous, or aspirational goal into an actionable uncertainty map where every node has a survival condition.

## The Core Idea

Standard AI planning optimizes for *looking helpful*. Decomposer optimizes for *being honest*.

The result is structurally different from any AI-generated plan you've seen: every claim has a survival condition, every gap is explicitly owned, and at least 30% of the map is explicitly uncertain.
